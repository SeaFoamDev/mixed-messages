# **Welcome to my random message generater!**
### What It Does:
1. Generate random messages, quotes, jokes, and fortunes!
2. Uses a index.js file as the source for the message!

In the furture I hope to add this to a HTML docuement, so it is more interactive, but for now you are going to have to run it through node. If you have any questions, new message ideas, or find a bug please contact me and I will get back to you as soon as possible. Enjoy!
